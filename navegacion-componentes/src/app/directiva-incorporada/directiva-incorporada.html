Directivas en angular > Directivas Incorporadas

Angular proporciona una serie de directivas incorporadas que son muy útiles para la manipulación del DOM y la gestión de la interacción con el usuario. Algunos ejemplos de directivas incorporadas son If, For, Switch, ngStyle y ngClass.

If

@if(5 > 4){

Soy un texto que se puede mostrar

}

For

@for(nombre of nombres; track nombre){

{{nombre}}

}

Switch

@switch (tipoMensaje) { @case ('aviso') {

Este es solo un aviso.

} @case ('error') {

Ha ocurrido un error.

} @default {

Este es un mensaje genérico.

} }
